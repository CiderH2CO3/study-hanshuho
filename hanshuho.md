# 1. 本人確認（KYC）の全体像

---

# 1. 本人確認（KYC）の全体像

### 顧客確認（CDD）の重要性
- **特定事業者**は、取引の開始時に顧客の本人特定事項を確認する義務があります。
- マネー・ローンダリングおよびテロ資金供与対策（AML/CFT）の基盤となります。

### 本人確認の三要素
1. **本人特定事項**: 氏名、住居、生年月日（自然人の場合）
2. **取引目的**: 取引を行う目的の確認
3. **職業・事業内容**: 顧客の属性確認

# 2. 準拠バージョンと改正予定

---

# 2. 準拠バージョンと改正予定

### 準拠している条文
- **犯罪による収益の移転防止に関する法律施行規則（第六条・第七条）**
- **2025年（令和7年）6月施行分まで反映済み**
  - 第六条第一項第一号「ル」（スマホ版マイナ）を含む最新の内容です。

### 改正のロードマップ（2025年〜2027年）

| 時期 | 主な変更点 |
|------|------------|
| **2025年6月** | **方式「ル」の新設**: スマホ版マイナ（カード代替電磁的記録）の利用開始。 |
| **2027年4月** | **大規模改正**: 券面撮影方式（ホ・ト(画像)・リ）等の廃止、IC読取の原則義務化。 |

---

# 2. 2027年4月 改正のポイント

2027年4月以降、**「券面の画像（厚み等）を送るだけ」の本人確認は原則廃止**され、ICチップ活用または原本送付が必須となります。

### 主な変更点
- **券面撮影方式の廃止**: 偽造対策のため、スマホカメラによる厚み撮影等を含む画像送信（現行ホ方式等）が認められなくなります。
- **ICチップ活用の義務化**: 対面・非対面を問わず、本人確認書類のICチップ情報の読み取りが原則必須となります。
- **方式記号の再編**: 条文の整理に伴い、大幅な記号の振り直しが行われます。
  - 例：公的個人認証（JPKI）が「ワ・カ」から「リ・ヌ」へ。

# 方式「リ」と「ヌ」の違い

郵送で「書類の写し（コピー）」を送る点は共通ですが、**「誰が」「どのような取引で」**使えるかが異なります。

| 項目 | 方式「リ」 | 方式「ヌ」 |
|:---|:---|:---|
| **対象者** | 一般の個人顧客 | 法人の被用者（従業員）等 |
| **必要書類** | 写し**2点**（または写し1点＋補完書類） | 写し**1点** |
| **前提条件** | 特になし | 特定の法人取引（給与振込口座等）に限る |
| **2027年改正** | **廃止**（新「ホ」方式へ） | **存置**（新「ヘ」方式へ） |

- **ポイント**: ヌ方式は、法人が身元をある程度保証している（給与振込等）ため、リ方式よりも簡素化（書類1点）されています。

# 方式「ワ」「カ」「ヨ」の違い

これらはすべて「電子署名」を利用するデジタル完結方式ですが、**「どの法律に基づく証明書か」**が異なります。

| 項目 | 方式「ワ」 | 方式「カ」 | 方式「ヨ」 |
|:---|:---|:---|:---|
| **通称** | **認定電子署名** | **公的個人認証 (JPKI)** | **特定電子署名** |
| **根拠法** | 電子署名法（認定） | 公的個人認証法 | 犯収法施行規則 |
| **発行体** | 民間認定事業者（セコム等） | J-LIS（地方公共団体） | 認定を受けた特定事業者 |
| **主な証明書** | 商業登記用電子証明書など | マイナンバーカード | 特定のeKYC用証明書 |
| **2027年改正** | **新「リ」方式**へ | **新「ヌ」方式**へ | **新「ル」方式**へ |

- **実務上の選択**: 一般個人向けには、普及率の高い**「カ（JPKI）」**が最も一般的です。
